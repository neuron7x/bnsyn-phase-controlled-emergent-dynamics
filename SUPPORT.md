# Support

## Getting help

- For usage questions, open a GitHub Discussion or Issue.
- For reproducibility or setup issues, include your command history and environment details.
- For security-sensitive reports, follow [SECURITY.md](SECURITY.md).

## Before opening a request

1. Check [README.md](README.md) for setup and quickstart commands.
2. Check [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md) for known issues.
3. Include exact commands and outputs when reporting a problem.

## Issue template hints

Helpful diagnostics:

- Python version
- OS and architecture
- Install command used
- Failing command and full stderr/stdout
