# Maintainers

## Current maintainers

The canonical maintainer list is the set of users with merge/admin access in the GitHub repository settings.

## Maintainer responsibilities

- Review and merge changes that satisfy repository quality gates.
- Keep release, governance, and evidence workflows operational.
- Coordinate issue triage and documentation updates.
