# Code of Conduct

## Purpose

This repository follows a respectful, collaborative working style for issues, pull requests, reviews, and discussions.

## Expected behavior

- Be respectful and professional.
- Focus feedback on code, data, and documentation.
- Use evidence from repository artifacts when discussing technical changes.

## Unacceptable behavior

- Harassment, hate speech, or personal attacks.
- Doxxing or publication of private information.
- Disruptive behavior that blocks constructive collaboration.

## Reporting

For conduct concerns, open a private security-style report using repository security/contact channels documented in [SECURITY.md](SECURITY.md).

## Scope

This policy applies to repository spaces including issues, pull requests, and discussions.
